## Установка
```bash
git clone https://github.com/EgorSavchuk/autoMOZI

cd autoMOZI

pip3 install -r requirements.txt
```

## Запуск с выводом в консоль
```bash
python3 autoMOZI.py
```

## Запуск с выводом в html файл
```bash
python3 autoMOZI.py -html
```
